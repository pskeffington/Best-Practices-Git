# Stage 06 Validation Report

## Primary performance

Report performance using the protocol-defined primary metric.

## Secondary performance

Report secondary metrics and explain what each adds.

## Calibration

Assess whether predicted probabilities or scores align with observed outcomes when applicable.

## Subgroup performance

Evaluate performance across substantively important groups when the data and research question support it.

## Error analysis

Describe false positives, false negatives, unstable cases, and systematic failure modes.

## Robustness checks

Report sensitivity analyses, alternative splits, alternative specifications, and stress tests.

## Interpretation boundary

State what the validation design supports and what it does not support.

## Exit gate

Proceed when the model has been evaluated in a way that supports the manuscript claims.
