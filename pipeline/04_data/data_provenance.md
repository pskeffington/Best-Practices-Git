# Stage 04 Data Provenance

## Data source

Name the dataset, owner, collection period, access conditions, and retrieval date.

## Data permissions

Document license, data-use agreement, privacy restrictions, and sharing limits.

## Unit of analysis

Define the observational unit and any repeated-measures structure.

## Inclusion and exclusion rules

Record every rule used to construct the analytic sample.

## Data dictionary

Create or link a data dictionary with variable names, labels, types, coding rules, and missing-value codes.

## Preprocessing log

Document transformations, derived variables, scaling, encoding, imputation, and quality checks.

## Leakage review

List variables or transformations that could introduce outcome leakage or future information.

## Exit gate

Proceed when the analytic dataset can be reconstructed or audited from documented steps.
