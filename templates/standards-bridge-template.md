# Standards Bridge Template

Use this template to connect a manuscript pipeline stage to reporting standards and final paper language.

## Pipeline stage

Name the scaffold stage.

## Scientific decision captured

Describe the scientific decision preserved by this stage.

## Reporting standards supported

List the standards, checklists, or transparency practices this stage supports.

## Required artifact

Name the artifact file that records the decision.

## Evidence generated

Describe the evidence, table, figure, or prose that this stage produces.

## Manuscript section supported

Identify the manuscript section where this evidence will appear.

## Reviewer-facing value

Explain how this stage helps a reviewer evaluate validity, reproducibility, or interpretation.

## Claim boundary

State what the manuscript is allowed to claim after this stage and what it should not claim.
