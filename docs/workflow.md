# Manuscript Workflow

This repository treats manuscript production as a staged research process.

## Stage 1: Query

Write the research question before selecting a model. Record the population, outcome, predictor space, use case, and contribution.

## Stage 2: Protocol

Define the data source, sampling frame, preprocessing plan, baseline model, candidate models, evaluation metrics, and validation design.

## Stage 3: Analysis

Run the workflow from source-controlled scripts. Save generated outputs outside Git unless they are lightweight and necessary for peer review.

## Stage 4: Interpretation

Compare results against the baseline. Identify failure modes, uncertainty, ethical concerns, and limits of generalization.

## Stage 5: Manuscript

Write the manuscript in LaTeX using modular section files. Keep claims aligned with the evidence produced by the analysis.

## Stage 6: Submission

Check journal formatting, citation style, reporting requirements, data availability statements, code availability statements, and reproducibility materials.
